How to use Hexo in Github Pages(1)

Setup blog generator

1. Create repositories in Github.

  • Create .github.io repository for the Github Pages.
  • Create another repository as the blog generator, I recommend using .github.io.hexo in this case.

2. Install Hexo locally

Make should you have NPM and Git installed in your Linux/Mac machine.

3. Create a Hexo project

Create a new fold to hold the Hexo code. The fold name does not matter, I use blog-generator in this case.

4. Install deployer for git

5. Update Hexo config

Use any editor or vi to update the config.yml, you could change a bunch of settings within this file. The most important configuration is the deploy

6. Deploy blog to Github Pages!

The hexo has generated the static blog files, and push all of them to your .github.io repository.
Open a browser with the URL .github.io to see your Github Pages.

7. Add blog generator to Git
